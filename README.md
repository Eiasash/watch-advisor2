# watch-advisor2

A fresh-start watch-first outfit planning app.

## Features
- seeded real watch collection
- watch rotation engine
- outfit scoring based on formality, color, strap harmony, weather, and diversity
- thumbnail-first garment import
- IndexedDB cache bootstrap
- background cloud sync hook
- virtualized wardrobe grid

## Run
```bash
npm install
npm run dev
```

Copy `.env.example` to `.env` if you want Supabase enabled.
